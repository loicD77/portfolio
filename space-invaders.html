Présentation du projet

Space Invaders est un jeu développé en Java dans le cadre de ma Licence 3 à Sorbonne Université. Ce projet a été conçu pour mettre en pratique mes compétences en programmation orientée objet et en création de jeux interactifs. Il s'inspire du célèbre jeu d'arcade des années 80.

Objectifs et fonctionnalités principales

  • Interface graphique : Création d'une interface utilisateur interactive avec Java Swing.
  • Déplacement dynamique : Gestion fluide des mouvements du vaisseau du joueur à l'aide des touches clavier.
  • Système de tir : Le joueur peut tirer sur les envahisseurs pour les détruire.
  • Ennemis autonomes : Les envahisseurs se déplacent en groupe, augmentant la difficulté au fil du temps.
  • Gestion des collisions : Détection précise des collisions entre les tirs du joueur et les ennemis.
  • Système de score : Un compteur affiche le score du joueur en temps réel.

Technologies utilisées

  • Langage : Java
  • Bibliothèque graphique : Java Swing
  • Paradigme : Programmation orientée objet (POO)
  • Outil de développement : IntelliJ IDEA

Développement et défis techniques

Durant le développement de Space Invaders, j'ai rencontré plusieurs défis techniques, notamment :

  • Gestion des événements clavier : Implémenter une détection fluide des touches pour garantir une bonne expérience utilisateur.
  • Optimisation des performances : Assurer une fluidité constante, même lorsque de nombreux éléments (ennemis et tirs) sont affichés à l'écran.
  • Gestion des collisions : Établir un système précis pour détecter les impacts entre tirs et ennemis.
