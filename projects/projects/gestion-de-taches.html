Planificateur de Tâches Annuel

Une application web de gestion des corvées et des tâches annuelles.

← Retour aux projets

Présentation du projet

Ce planificateur de tâches permet de gérer efficacement les corvées et tâches récurrentes sur une année complète. Il est conçu pour simplifier la gestion et l'organisation des tâches.

Fonctionnalités principales

  • 🔹 Création, mise à jour et suppression des tâches.
  • 🔹 Système de planification annuelle.
  • 🔹 Stockage et gestion des données avec MongoDB Atlas.
  • 🔹 Interface utilisateur intuitive et ergonomique.
  • 🔹 API backend sécurisée avec gestion des utilisateurs.

Technologies utilisées

🌐 Frontend : HTML, CSS, JavaScript 💾 Base de données : MongoDB Atlas 🖥️ Backend : Node.js, Express 🔐 Authentification : JWT
