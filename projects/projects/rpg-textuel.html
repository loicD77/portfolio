Présentation du projet

RPG Textuel est un jeu de rôle interactif développé en Java, jouable via la console. Ce projet a été réalisé dans le cadre de ma Licence 3 afin d'explorer les concepts de programmation orientée objet et de design de systèmes interactifs.

Fonctionnalités principales

  • Création de personnages : Le joueur peut personnaliser son personnage en choisissant son nom et ses attributs (force, agilité, intelligence).
  • Combat dynamique : Système de combat au tour par tour contre des ennemis générés aléatoirement.
  • Système d'inventaire : Gestion des objets ramassés, équipés, ou consommés.
  • Exploration : Parcours de différentes zones, avec des événements aléatoires pour enrichir l'expérience.
  • Progression : Gain d'expérience et montée de niveaux pour améliorer les compétences du personnage.
  • Économie virtuelle : Système de monnaie virtuelle permettant l'achat et la vente d'objets dans des magasins.

Technologies utilisées

  • Langage : Java
  • Paradigme : Programmation orientée objet (POO)
  • IDE : IntelliJ IDEA
  • Conception : Utilisation de diagrammes UML pour modéliser les classes et les interactions.

Développement et défis techniques

Ce projet m'a permis de relever plusieurs défis techniques :

  • Structure modulaire : Organisation du code en plusieurs classes pour représenter les personnages, ennemis, objets, et zones.
  • Gestion des interactions : Création de menus interactifs pour guider le joueur dans ses choix.
  • Systèmes de combat : Mise en œuvre d'algorithmes pour les attaques, la défense, et la gestion des points de vie.
  • Randomisation : Génération aléatoire d'événements pour rendre chaque partie unique.
